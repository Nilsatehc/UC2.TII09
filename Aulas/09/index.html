Introdução á tag < img >

A tag < img> em html é usada para incorporar imagens em uma página web. Ela é um elemento vazio, ou seja, não possui uma tag de fechamento e contém apenas atributos quedefinem o conteudo visual.

Atributos principais

  • src (source): Define o caminho de uma imagem que será exibida. Pode ser uma URL exterma ou um caminho relativo dentro do servidor.

    Exemplo 1:

    <img src="image01.jpg">

  • alt (alternative text): fornece um texto alternativo para ser exibido caso a imagem não possa ser carregada. Ele também melhora a acessibilidade para leitores de tela.

    <"img src="gg.gif">

Tipos comum de imagens

Os formatos de imagens mais utilizados na web são:

  • .jpg ou .jpegFormato utilizado amplamente para fotografias e imagens com muitos detalhes e cores. Possui compressão com perda de qualidade (lossy).
  • (.png): Usado para imagens com fundo transparente ou que necessitam de alta qualidade, sem perda de detalhes. Utiliza compressão sem perda (lossless).
  • GIF (.gif): Formato ideal para imagens animadas e pequenos ícones. Suporta transparência, mas tem um número limitdo de cores.
