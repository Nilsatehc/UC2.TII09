HTML (linguagem de marcação de hipertexto) é o bloco e construção mais básico da web.Define o significado e a assinatura de conteúdo web

TAG ou Marcação

O HTML usa "Marcação" para anotar texto,imagem e outros conteúdos para exibção em um navegador web

Um elemento HTML é separado do outro texto em um documento por "tags". As tags consistem no nome do elemento entre um "<" e ">".
